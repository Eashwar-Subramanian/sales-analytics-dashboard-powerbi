---

## 🚩 Problem Statement  
AtliQ Hardware struggles with inconsistent sales reports and lacks real-time insights.

---

## 🎯 Project Planning (AIMS GRID)

- **Purpose:** Deliver timely, accurate sales insights via dashboard  
- **Stakeholders:** Sales director, marketing, IT, analytics team  
- **End Result:** Automated Power BI dashboard showing latest sales trends  
- **Success Criteria:** 1) Accurate sales insights; 2) 10% cost saving for sales team  

---

## 🛠️ Technologies Used

- Data Storage: MySQL Database  
- Data Processing: Power BI (ETL, Data Cleaning, Visualization)  
- Techniques: Data Warehousing, Data Munging/Wrangling  

---

## 📊 Project Workflow

1. Extracted sales, customer, product, and market data from MySQL  
2. Performed ETL and data cleaning in Power BI  
3. Built interactive dashboard to display 4 years sales trends  
4. Incorporated stakeholder feedback for iterative improvements  

---

## 📈 Key Results

- Dashboard visualized AtliQ's last 4 years of sales trends  
- Enabled better understanding and informed decisions  
- Expected to boost revenue by 7% next quarter
